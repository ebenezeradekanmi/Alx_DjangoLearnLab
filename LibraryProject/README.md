# LibraryProject

## Objective
Gain familiarity with Django by setting up a Django development environment and creating a basic Django project.

## Steps
1. Installed Django using `pip install django`.
2. Created a new Django project named `LibraryProject` using `python -m django startproject LibraryProject`.
3. Navigated into the project directory.
4. Started the development server using `python manage.py runserver`.
5. Visited `http://127.0.0.1:8000/` to view the default Django welcome page.
6. Applied initial migrations using `python manage.py migrate`.
7. (Optional) Created a Django superuser using `python manage.py createsuperuser`.
